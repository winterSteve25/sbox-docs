# Download & Install

The s&box editor and game are available [on Steam](https://store.steampowered.com/app/590830/sbox/).

## Install

![Launch the editor from your Steam library](./images/launch-the-editor-from-your-steam-library.png)

To install s&box [click here](steam://run/2129370) or:

1. Open Steam
2. Click on the Library tab
3. Search for s&box
4. Install both the game and the editor apps

## Launch

You can launch s&box in several ways:

- Launch through Steam directly
- Use a shortcut to `sbox-dev.exe`
- Open your `.sbproj` file directly
